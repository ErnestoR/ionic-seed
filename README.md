Ionic Application Seed
=====================

A starting project for all Ionic V1 projects. Features:

- ES6 with Babel
- Webpack
- ngAnnotate
- SCSS imports
- **Proper** application architecture
- Testing harness with Karma

## Using this project

```bash
$ git clone --depth 1 <url>
$ npm i -g ionic
$ npm i
```
